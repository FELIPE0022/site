1.1 Componentes Internos

1.1.1 CPU

A CPU (Unidade Central de Processamento) é o cérebro do computador, responsável pelo processamento das informações.

1.1.2 Armazenamento

Os dispositivos de armazenamento, como HDs e SSDs, guardam dados e programas permanentemente.

1.1.3 Memória

A memória RAM armazena temporariamente dados que estão sendo processados pelo computador.

1.1.4 Fonte de Alimentação

A fonte de alimentação converte energia elétrica para fornecer energia adequada aos componentes do computador.

1.2 Componentes Externos

1.2.1 Teclado

Dispositivo de entrada utilizado para digitação de informações no computador.

1.2.2 Monitor de Vídeo

Dispositivo de saída responsável por exibir as informações processadas pelo computador.

1.2.3 Mouse

Dispositivo de entrada utilizado para controlar o cursor na tela.

2. Grandezas Elétricas

2.1 Potência Elétrica

Quantidade de energia elétrica consumida ou gerada por um dispositivo em um intervalo de tempo.

2.2 Tensão Elétrica

Diferença de potencial elétrico entre dois pontos de um circuito.

2.3 Corrente Elétrica

Fluxo de elétrons através de um condutor, medido em ampères.
